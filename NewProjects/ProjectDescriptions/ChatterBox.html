ChatterBox

When I wrote my resume, I realized that I had skills in web development with nothing to show, so I decided it would be appropriate to design a personal web portfolio--one that could showcase my ability to create websites.

I wrote this website using Ruby on Rails, HTML, CSS, and JS. The app was deployed with Heroku and uses PostgreSQL for database design. It contains CRUDing that is gated by an administrative login.
